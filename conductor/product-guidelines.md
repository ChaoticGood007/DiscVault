# Product Guidelines: DiscVault

## Prose Style
- **Tone:** Enthusiastic & Approachable. We want to celebrate the sport of disc golf and make the tool feel like it's built by enthusiasts for enthusiasts.
- **Language:** Clear, helpful, and community-focused. Use accessible language while maintaining professional precision.

## UX Principles
- **Visual-First Aesthetic:** Discs are beautiful objects. Use rich imagery (where possible), high-quality icons, and interactive charts to make the data feel alive.
- **Data-Rich Efficiency:** While visual, the tool must provide dense, actionable information. Expert users should be able to perform advanced filtering and bulk actions with ease.
- **Mobile-First Design:** Players should be able to manage their bags in the field. All core features must be fully functional and easy to use on mobile devices.

## Branding
- **Modern Utility:** The brand should feel clean, modern, and serious—a tool you can trust with your most valuable collections. Use a sophisticated "Slate" palette with a forced light theme for clarity and a professional feel.

## Development Priorities
- **Performance First:** Search results and inventory lists must load and filter instantly, even with 1,000+ items. Infinite scroll and efficient data fetching are non-negotiable.
- **Robust Reliability:** Data integrity is paramount. Ensure all inventory changes are atomic and well-validated. Prevent data loss at all costs.
