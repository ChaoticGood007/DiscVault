# Technology Stack: DiscVault

## Languages & Core Frameworks
- **Language:** TypeScript
- **Framework:** Next.js 16 (App Router)
- **Library:** React 19

## Database & Data Layer
- **Database:** SQLite (Self-hosted)
- **ORM:** Prisma 6

## Styling & UI Components
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide Icons
- **Animations:** Framer Motion
- **Charts:** Recharts (for collection analytics)

## Project Standards
- **Component Style:** Modern, functional components using Server Actions for data mutations.
- **Data Integrity:** Strict Prisma typing and validation for all inventory entries.
